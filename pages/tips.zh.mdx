# 写作指南

这里会列举一些主题推荐的写作指南和注意事项

## 标题

请不要在 Markdown 文章开头加上 `# 标题`

请**仅在** Markdown 文件的 `front-matter` 中的 `title: ` 属性中写上你的标题

```markdown {2}
---
title: 主题样式 Demo
date: 2022-10-02 19:07:05
tags: 
    - "Demo"
    - "Hexo"
    - "Hexo Theme"
    - "Hexo Theme Redefine"
thumbnail: https://evan.beee.top/img/redefine-1-final.webp
sticky: 999
---
```

**否则**，会出现这种情况
