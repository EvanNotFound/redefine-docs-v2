# Project Introduction



![Slogan](https://evan.beee.top/img/2023/03/18/610599507c8078e7279966455688c66c.webp)

**Redefine** is a clean and fast hexo theme. Simple, but not simplistic. This theme includes many widely used features.

This theme is developed based on [hexo-theme-keep](https://github.com/XPoet/hexo-theme-keep), thanks to XPoet for the open-source contribution. This theme mainly optimizes the style, adds a writing module, and supports various plugins. At the same time, it increases the level of customization, allowing you to use this theme more conveniently.
