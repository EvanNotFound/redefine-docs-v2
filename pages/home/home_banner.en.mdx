## Subtitle

### text content

**Configuration item name: `home_banner.subtitle.text`**

There is a typewriter animation effect, and multiple strings can be configured, and each string will be displayed cyclically in the typewriter animation.

| Type | Remarks | Example |
| :---: | :---: | :---: |
| array | separated by commas | `["This is the first message", "This is the second message", "etc"]` |

### Advanced: Typing animation settings

- `typing_speed`: typing speed (number of letters printed per millisecond)
- `backing_speed`: Backspace speed (number of letters backspaced per millisecond)
- `starting_delay`: start delay (milliseconds)
- `backing_delay`: backspace delay (milliseconds)
- `loop`: Whether to loop
- `smart_backspace`: whether to backspace intelligently

### Link

**Configuration item name: `home_banner.social_links.links`**

Please add your social link, the format is as follows, please fill in the Brands icon name, such as `github`.

```yaml filename="_config.redefine.yml"
links:
     [fontawesome icon name (not HTML code)]: [your social link address]
```

You can go to [Font Awesome](https://fontawesome.com/search?o=r&f=brands) to check the icon name.
