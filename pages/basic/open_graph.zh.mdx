# Open Graph

详细信息请参考 [Open Graph protocol](https://ogp.me/)。

## 开关

**配置项名称：`open_graph.enable`**

| 类型 | 可选值 | 默认值 |
| :---: | :---: | :---: |
| 布尔值 | `true \| false` | `false` |

## OG 图片

**配置项名称：`open_graph.image`**

Open Graph 图片，用于分享到社交媒体时的预览图。

| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 本地图片路径/图片链接 | `images/og.png` |

## OG 描述 `description`

Open Graph 描述，用于分享到社交媒体时的预览描述。

| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 描述 | `Theme Redefine is a simple and elegant theme for Hexo.` |
