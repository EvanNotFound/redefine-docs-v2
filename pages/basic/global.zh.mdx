# 全局功能设置 `global`

设置全站的一些功能。

## 自定义字体 `fonts`

可以自定义全局字体

### 中文字体 `chinese`

#### 开关 `enable`

请填写 `true` 或 `false`

#### 名称 `family`

请填写字体名称，例如 `Noto Sans SC`

#### 链接 `url`

请填写字体样式文件地址（CSS），比如 Google Font 中的 `https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;700&display=swap`

### 英文字体 `english`

#### 开关 `enable`

请填写 `true` 或 `false`

#### 名称 `family`

请填写字体名称，例如 `Farsan`

#### 链接 `url`

请填写字体样式文件地址（CSS），比如 Google Font 中的 `https://fonts.googleapis.com/css2?family=Farsan&display=swap`

## 文章最大宽度 `content_max_width`

单位为 `px`，默认为 `1000px`。如无特别需要请不要修改。

## 边栏宽度 `sidebar_width`

单位为 `px`，默认为 `210px`。如无特别需要请不要修改。

## 悬停效果 `hover`

鼠标悬停在各种模块（文章卡片，文章列表，功能卡片等）时的效果。

### 阴影效果 `shadow`

请填写 `true` 或 `false`，默认为 `true`。

### 缩放效果 `scale`

请填写 `true` 或 `false`，默认为 `false`。

## 网站统计 `busuanzi_counter`

- `enable` 是否启用，填写 `true` 或 `false`，默认为 `true`。
- `site_pv` 是否显示网站总访问量，填写 `true` 或 `false`，默认为 `true`。
- `site_uv` 是否显示网站总访客数，填写 `true` 或 `false`，默认为 `true`。
- `post_pv` 是否显示文章总访问量，填写 `true` 或 `false`，默认为 `true`。

## PJAX 加载 `pjax` 

请填写 `true` 或 `false`，默认为 `true`。

PJAX 能够使网页加载更快，有更好的用户体验。
