# 全局功能设置 `global`

## 网站统计

**配置项名称：`global.website_counter`**

- `enable` 是否启用，填写 `true` 或 `false`，默认为 `true`。
- `site_pv` 是否显示网站总访问量，填写 `true` 或 `false`，默认为 `true`。
- `site_uv` 是否显示网站总访客数，填写 `true` 或 `false`，默认为 `true`。
- `post_pv` 是否显示文章总访问量，填写 `true` 或 `false`，默认为 `true`。

## 单页应用体验

**配置项名称：`global.single_page`**

使用 Swup 库实现单页应用体验，类似 Pjax 但是比 Pjax 使用更方便，推荐开启。其他详细信息请参考 [Swup](https://swup.js.org/)。

| 类型 | 可选值 | 默认值 |
| :---: | :---: | :---: |
| 布尔值 | `true \| false` | `true` |

由于浏览器不会刷新，所以在 markdown 文件里面加入了自定义 javascript 的话，请在你的 `script` 标签中加入 `data-swup-reload-script` 属性，例如：

```html
<script data-swup-reload-script type="text/javascript">
  // 你的脚本
</script>
```
