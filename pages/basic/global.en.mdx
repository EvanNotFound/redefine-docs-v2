# 全局功能设置 `global`

设置全站的一些功能。

## 自定义字体

### 中文字体开关


**配置项名称：`global.chinese.enable`**

| 类型 | 可选值 | 默认值 |
| :---: | :---: | :---: |
| 布尔值 | `true \| false` | `false` |

### 中文字体名称

**配置项名称：`global.chinese.family`**

| 类型 | 示例 |
| :---: | :---: |
| 字符串 | `Noto Sans SC` |

### 中文字体链接

**配置项名称：`global.chinese.url`**



| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 字体样式文件地址（CSS） | `https://fonts.googleapis.com/css2?family=Noto+Sans+SC:wght@400;500;700&display=swap` |

## 网站统计

**配置项名称：`global.busuanzi_counter`**

- `enable` 是否启用，填写 `true` 或 `false`，默认为 `true`。
- `site_pv` 是否显示网站总访问量，填写 `true` 或 `false`，默认为 `true`。
- `site_uv` 是否显示网站总访客数，填写 `true` 或 `false`，默认为 `true`。
- `post_pv` 是否显示文章总访问量，填写 `true` 或 `false`，默认为 `true`。

## PJAX 加载

**配置项名称：`global.pjax`**

PJAX 能够使网页有单页应用的体验，不需要刷新页面就能加载新的内容。推荐开启。

| 类型 | 可选值 | 默认值 |
| :---: | :---: | :---: |
| 布尔值 | `true \| false` | `true` |
