# Homepage Thumbnail

If you want to add a thumbnail to the homepage articles, like this:

![Screen Shot 2023-01-05 at 8.30.26 AM](https://evan.beee.top/img/Screen%20Shot%202023-01-05%20at%208.30.26%20AM.png)

## Configuration Process

Add the following to the `front-matter` of the corresponding article:

```yaml
thumbnail: "IMAGE_LINK"
```

## Example

For the configuration of the screenshot I provided above:

```markdown
---
title: Theme Demo
date: 2022-12-30 19:07:05
tags: "demo"
thumbnail: "https://evan.beee.top/img/208184324-f2640ade-587a-4f46-8ad1-7b4c1b31394f.png"
---
```
