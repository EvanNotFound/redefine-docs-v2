# Home page summary

## Custom Summary

If you want to customize the article summary of the home page

![Screen Shot 2022-12-20 at 4.42.04 PM](https://evan.beee.top/img/Screen%20Shot%202022-12-20%20at%204.42.04%20PM.png)

Configuration process:

1. Add `excerpt:"summary"` attribute to the [front matter](https://hexo.io/docs/front-matter.html) of the article page, so far, you can see that the summary of the home page has changed

    Example:

    ```markdown
    ---
    title: Excerpt Test
    date: 2022-12-20 12:12:12
    tags: Excerpt
    categories: Excerpt
    excerpt: "This is the excerpt of the post"
    ---
    ```

## Close article summary

If you want to close the article summary

![Screen Shot 2022-12-23 at 11.21.21 AM](https://evan.beee.top/img/Screen%20Shot%202022-12-23%20at%2011.21.21%20AM.png)

Configuration process:

Add the attribute `excerpt: false` to the [front matter](https://hexo.io/docs/front-matter.html) of the article page, so far, you can see that the summary on the home page disappears

Example:

```markdown
---
title: Excerpt Test
date: 2022-12-20 12:12:12
tags: Excerpt
categories: Excerpt
excerpt: false
---
```
