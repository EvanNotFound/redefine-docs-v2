# Footer

## Runtime

**Configuration item name: `footer.runtime`**

| Type | Possible Values | Default Value |
| :---: | :---: | :---: |
| Boolean | `true \| false` | `true` |

![image-20230322200032581](https://evan.beee.top/img/2023/03/22/c6680e3c7d99417e4ca383eb80e23222.png)

## Icon

**Configuration item name: `footer.icon`**

| Type | Note | Example |
| :---: | :---: | :---: |
| String | Please fill in Fontawesome HTML code | `<i class="fa-solid fa-heart fa-beat" style="--fa-animation-duration: 0.5s; color: #f54545"></i>` |

For Fontawesome icons with animation, please refer to the [Fontawesome Animated Icons Documentation](https://fontawesome.com/docs/web/style/animate).

## Site Start Time

**Configuration item name: `footer.start`**

| Type | Note | Example |
| :---: | :---: | :---: |
| String | Please fill in the time format `YYYY/MM/DD HH:mm:ss` | `2022/8/17 11:45:14` |

## Custom Footer Information

**Configuration item name: `footer.customize`**

| Type | Note | Example |
| :---: | :---: | :---: |
| String | Supports HTML code | `You're right, but "Genshin Impact" is a brand new open-world adventure game developed by miHoYo.` |

## ICP Record Number

### Switch

**Configuration item name: `footer.icp.enable`**

| Type | Possible Values | Default Value |
| :---: | :---: | :---: |
| Boolean | `true \| false` | `false` |

### Record Number

**Configuration item name: `footer.icp.number`**

| Type | Note | Example |
| :---: | :---: | :---: |
| String | Please fill in the ICP record number | `粤ICP备2021000000号` |

### Custom Record Link

**Configuration item name: `footer.icp.url`**

If not filled, it will automatically redirect to https://beian.miit.gov.cn/.

| Type | Note | Example |
| :---: | :---: | :---: |
| String | Optional | `https://icp.gov.moe/` |
