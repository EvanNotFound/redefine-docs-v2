# footer

## operation hours

**Configuration item name: `footer.runtime`**

| type | optional value | default value |
| :---: | :---: | :---: |
| Boolean | `true \| false` | `true` |

## icon

**Configuration item name: `footer.icon`**

| Type | Remarks | Example |
| :---: | :---: | :---: |
| String | Please fill in Fontawesome HTML code | `<i class="fa-solid fa-heart fa-beat" style="--fa-animation-duration: 0.5s; color: #f54545"></i> ` |

For animated Fontawesome icons, please refer to [Fontawesome Animated Icons Documentation](https://fontawesome.com/docs/web/style/animate).

## Build time

**Configuration item name: `footer.start`**

| Type | Remarks | Example |
| :---: | :---: | :---: |
| String | Please fill in the time in `YYYY/MM/DD HH:mm:ss` format | `2022/8/17 11:45:14` |

## Custom footer information

**Configuration item name: `footer.customize`**

| Type | Remarks | Example |
| :---: | :---: | :---: |
| String | Support HTML code | `You are right, but "Yuanshinshin" is a new open world adventure game independently developed by MiHoYo. ` |

## ICP record number

### switch

**Configuration item name: `footer.icp.enable`**

| type | optional value | default value |
| :---: | :---: | :---: |
| Boolean | `true \| false` | `false` |

### case number

**Configuration item name: `footer.icp.number`**

| Type | Remarks | Example |
| :---: | :---: | :---: |
| String | Please fill in the ICP record number | `Guangdong ICP No. 2021000000` |

### Custom filing link

**Configuration item name: `footer.icp.url`**

If it is not filled in, it will automatically jump to https://beian.miit.gov.cn/.

| Type | Remarks | Example |
| :---: | :---: | :---: |
| string | optional | `https://icp.gov.moe/` |
