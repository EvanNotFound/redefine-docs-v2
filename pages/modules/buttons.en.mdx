## Writing

```
{% btn [optional size]::[name]::[url]::[optional icon] %}
```

For example:

```markdown
{% btn button without any parameters:: / %} is suitable for integrating into paragraphs.

The regular button is suitable for being independent of paragraphs:

{% btn regular::Example blog::https://www.ohevan.com::fa-solid fa-play-circle %}

{% btn regular::Example blog::https://www.ohevan.com::fa-solid fa-play-circle %}

The large button has a stronger emphasis and is recommended to be used with center:

{% btn center large::Get started::https://redefine-docs.ohevan.com::fa-solid fa-download %}
```

## Variable optional value

`[optional size]` :

```markdown
center, regular, large, center large, center regular
```

`[optional icon]` :

[Fontawesome](https://fontawesome.com/search) icon name, such as `fa-solid fa-house`
