# insert custom code

**Configuration item name: `inject`**

This function allows you to insert custom codes into the page, such as various CSS/JS codes such as statistical codes and click styles.

## configuration

In the `inject` configuration item of the Redefine theme configuration file `_config.redefine.yml`, you can configure the following:

* `enable`: Whether to enable this function, `true` is enabled, `false` is disabled.
* `head`: the code inserted into the head of the page, such as statistics code, etc.
* `footer`: the code inserted at the bottom of the page, such as click styles, etc.
