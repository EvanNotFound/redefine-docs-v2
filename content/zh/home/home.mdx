**配置项名称：`home.sidebar.links`**

写法格式请参考[导航栏](/home/navbar)。

留空则不显示。

示例配置：

```yaml
home:
  sidebar:
    links:
      Archives: # 显示的名称
        path: /archives # 路径
        icon: fa-regular fa-archive # 图标，可以为空
      Tags: # 显示的名称
        path: /tags # 路径
        icon: fa-regular fa-tags # 图标，可以为空
      Categories: # 显示的名称
        path: /categories # 路径
        icon: fa-regular fa-folder # 图标，可以为空
```

## 文章日期格式

<Image
  src="https://evan.beee.top/img/2023/03/22/6e5dd768ca4967820c295ed4819eae2c.png"
  alt="image-20230322184235442"
  width={2944}
  height={1118}
  className="my-2"
  style={{
    maxHeight: "500px",
    objectFit: "contain",
  }}
/>

**配置项名称：`home.article_date_format`**

|  类型  |                                    可选值                                    | 默认值 |
| :----: | :--------------------------------------------------------------------------: | :----: |
| 字符串 | `auto`, `relative`, `YYYY-MM-DD`, `YYYY-MM-DD HH:mm:ss`, `momentjs 日期格式` | `auto` |

- `auto`：自动，根据文章日期自动选择日期格式'
- `relative`：相对时间，如 `1 hour ago`
- `YYYY-MM-DD`：年月日，如 `2023-03-22`
- `YYYY-MM-DD HH:mm:ss`：年月日时分秒，如 `2023-03-22 12:00:00`
- `MMM D, YYYY`：月日年，如 `Mar 22, 2023`
- 更多格式请参考 [Moment.js](momentjs.com/docs/#/displaying/format/)
