## 写法

```markdown
{% folding [颜色]::[标题] %}

需要写的内容

{% endfolding %}
```

比如上图例子

```markdown
{% folding blue::Folding 测试： 点击查看更多 %}

啊啊啊啊啊

{% note danger  %}
danger 提示块标签
{% endnote %}

{% note tip  %}
tip 提示块标签
{% endnote %}

{% endfolding %}
```
