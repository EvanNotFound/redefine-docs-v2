## 运行时间

**配置项名称：`footer.runtime`**

| 类型 | 可选值 | 默认值 |
| :---: | :---: | :---: |
| 布尔值 | `true \| false` | `true` |

![image-20230322200032581](https://evan.beee.top/img/2023/03/22/c6680e3c7d99417e4ca383eb80e23222.png)

## 图标

**配置项名称：`footer.icon`**

| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 请填写 Fontawesome HTML 代码 | `<i class="fa-solid fa-heart fa-beat" style="--fa-animation-duration: 0.5s; color: #f54545"></i>` |

如需带动画的 Fontawesome 图标，请参考 [Fontawesome 动画图标文档](https://fontawesome.com/docs/web/style/animate)。

## 建站时间

**配置项名称：`footer.start`**

| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 请填写格式为 `YYYY/MM/DD HH:mm:ss` 的时间 | `2022/8/17 11:45:14` |

## 自定义页脚信息

**配置项名称：`footer.customize`**

| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 支持 HTML 代码 | `你说的对，但是《原神》是由米哈游自主研发的一款全新开放世界冒险游戏。` |

## ICP 备案号

### 开关

**配置项名称：`footer.icp.enable`**

| 类型 | 可选值 | 默认值 |
| :---: | :---: | :---: |
| 布尔值 | `true \| false` | `false` |

### 备案号

**配置项名称：`footer.icp.number`**

| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 请填写 ICP 备案号 | `粤ICP备2021000000号` |

### 自定义备案链接

**配置项名称：`footer.icp.url`**

不填写则会自动跳转至 https://beian.miit.gov.cn/。

| 类型 | 备注 | 示例 |
| :---: | :---: | :---: |
| 字符串 | 可选 | `https://icp.gov.moe/` |
