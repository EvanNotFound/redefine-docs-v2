---
title: Color Configuration (colors)
---

## Primary Color

**Configuration Item Name: `colors.primary`**

Please write the Hex value, such as `#1890ff`.

The primary color will affect the color of buttons, progress bars, links, and selected components.

## Secondary Color

**Configuration Item Name: `colors.secondary`**

Please write the Hex value, such as `#1890ff`.

The current version has not used it yet, but it is expected to be used in the future.
