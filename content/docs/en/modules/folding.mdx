You can nest other modules, text, or paragraphs under this module.

This module is partially migrated from `hexo-theme-volantis`.

## Effect

![Screenshot 2023-01-26 at 9.34.14 PM](https://evan.beee.top/img/2023/01/26/ade3c1205a969a2f69f1327225166288.png)

## Syntax

```markdown
{% folding [颜色]::[标题] %}

Content

{% endfolding %}
```

For example, in the above image:

```markdown
{% folding blue::Folding 测试： 点击查看更多 %}

啊啊啊啊啊

{% callout danger %}
danger callout content
{% endcallout %}

{% callout tip %}
tip callout content
{% endcallout %}

{% endfolding %}
```
