"Redefine" is a clean, fast, and pure Hexo theme. Its minimalist beauty is far
from ordinary, allowing you to feel the power of design. The theme includes
practical, rich features and a refined design philosophy.

The theme is derived from an early version of [hexo-theme-keep](https://github.com/XPoet/hexo-theme-keep) and has been carefully refined and expanded. While retaining the original theme's concise foundation, we optimized the overall styling, introduced rich writing modules, and added support for more plugins. We also improved customization flexibility so you can build a more distinctive personal blog.
